# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.3] — 2026-05-07

### Изменено
- В `auto`-режиме при конвертации декларативного блока в активный
  содержимое блока теперь полностью заменяется на отрендеренный шаблон
  `options.text` (раньше оставался старый текст «Нажимая кнопку…», что
  юридически противоречило наличию чекбокса).
- URL ссылки в блоке сохраняется из исходной разметки, если она там была;
  иначе используется `options.policyUrl`.
- Tooltip теперь использует `position: fixed` и `z-index: 2147483647`,
  плюс при показе автоматически поднимается выше любого видимого элемента
  под якорем (через `elementsFromPoint`). Это решает проблему перекрытия
  tooltip модалками с высоким z-index.

## [1.0.2] — 2026-05-07

### Исправлено
- В `auto`-режиме при найденном декларативном блоке («Нажимая кнопку,
  вы соглашаетесь…») теперь действительно вставляется активный
  `<input type="checkbox">` и к нему привязывается блокировка submit-кнопок.
  Раньше библиотека только правила/добавляла ссылку, оставляя блок пассивным.

### Добавлено
- Опция `autoInjectCheckbox` (по умолчанию `true`) — управляет вставкой
  чекбокса в найденный декларативный блок.
- Класс `cc-converted-block` на блоке после конвертации и
  `cc-injected-input` на инжектированном `<input>` — для возможной
  кастомной стилизации.

## [1.0.0] — 2026-05-07

### Добавлено
- Первый публичный релиз.
- Четыре режима вставки: `auto`, `append`, `after-submit`, `tooltip`.
- Auto-режим с поиском существующего блока согласия по ключевым словам.
- Подмена `href` у политики конфиденциальности в найденном блоке.
- Автоматическая блокировка submit-кнопок (включая `form="id"`-привязки).
- Перехват `submit` в capture-фазе (защита от программного `form.submit()`).
- `MutationObserver` для динамически добавленных форм.
- Tooltip-режим с отслеживанием видимости через `IntersectionObserver`.
- UMD-обёртка (AMD / CommonJS / global).
